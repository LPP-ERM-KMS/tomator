Welcome to tomator's documentation!
===================================
**tomator 1D** is open source, EUPL-licensed software written primarely in C++.
It is a 1-dimensional reaction-diffusion-convection code which describes plasma production
by RF waves inside a tokamak making use of the Braginskii continuity and heat balance equations.
